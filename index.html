KMS Superstore Dashboard

This dashboard, created using Tableau, shows sales and profit data for KMS superstore. It includes several key metrics such as total sales and total profit per customer segment, sum of profit per product category, sum of order quantity by province, etc.

The dashboard is designed to be easy to read and navigate, with clear labels and explanations for each metric. Users can filter the data by date range, province, and product category, allowing them to drill down into specific areas of interest.

COVID-19 Infection and Death Dashboard

This dashboard displays the global spread of the COVID-19 virus, including a forcest 2 years into the future. Using data from the World Health Organization, I was able to acheive this using my knowledge of MS SQL to clean the dataset and get the needed information to perform the visualization which was done using Tableau. The visualization this dashboard displays key metrics such as total cases and deaths. I hope this dashboard provides valuable information to those seeking to understand the impact of the pandemic.

Please note that the data shown on this dashboard is subject to change as the situation with the COVID-19 pandemic evolves. While every effort has been made to ensure the accuracy of this data, I am not responsible for any errors or omissions.
